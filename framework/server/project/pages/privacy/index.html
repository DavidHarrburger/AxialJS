Politique de confidentialité

Mise à jour le dimanche 22 Septembre 2024 - Site et données hébergés en France

Objectif

Conformément à la loi en vigueur et notamment au Règlement Général sur la Protection des Données (RGDP), la politique de confidentialité a pour objectif de vous informer sur :

  • • Le fonctionnement de ce site
  • • Les données personnelles recueillies
  • • L'utilisation des données personnelles recueillies
  • • Vos droits en tant qu'utilisateur de ce site

1. Fonctionnement du dndev.fr

1.1 - Hébergement et adresse du site

L'adresse de ce site est https://dndev.fr. Le site est hébergé en France par l'hébergeur Gandi.net / Gandi SAS :

  • 63, 65 Boulevard Massena
  • 75013 Paris
  • France
  • https://www.gandi.net/fr

1.2 - Hébergement des données

Les données personnelles recueillies ainsi que les données nécessaires au fonctionnement du site sont hébergée en France. Les bases de données utilisées sont fournies par MongoDB et l'hébergement de ces bases est fourni par AWS.

1.3 - Cookies, LocalStorage et SessionStorage

Les navigateurs (c'est-à-dire les logiciels qui vous permettent de naviguer sur Internet comme Google Chrome, Microsoft Edge, Mozilla Firefox ou encore Apple MacOs ou IOS Safari pour ne citer que les principaux) utilisent des API (Application Programming Interface - autrement dit du code) permettant de stocker / sauvegarder des informations sur votre ordinateur.

Parmi ces API, nous utilisons les Cookies, LocalStorage et SessionStorage lorsque vous naviguer sur ce site. Ces API sont utilisées uniquement à des fins de sécurité, lorsque vous vous identifiez pour accéder à votre page "client".

1.3 - Statistiques et analyse

Lorsque que vous naviguer sur ce site, nous recueillons des données statistiques et de mesure d'audience conformément au cadre établi par la CNIL :

https://www.cnil.fr/fr/mesurer-la-frequentation-de-vos-sites-web-et-de-vos-applications

Ce site utilise son propre système de mesure d'audience autrement dit, nous n'utilisons pas d'outils tels que Google Analytics, Matomo, Piwik Pro ou autre.

2. Données personnelles recueillies

Dans le cadre de la relation commerciale entre D n' Dev et vous-même, certaines données personnelles peuvent être recueillies et stockées dans une base de données.

Ces données peuvent être recueillies :

  • Sur la page de contact à l'adresse https//dndev.fr/contactvia le formulaire. Sont recueillis et cryptes votre nom, votre prénom, votre adresse mail, votre message.
  • Lors de la signature d'un contrat de services, nous intégrons les données personnelles et professionnelles dans notre base de données afin de vous fournir un portail de services et de suivi de votre développement. Vous avez la possibilité de décliner l'accès à ce service.

Dans tous les cas :

  • Toutes les données à caractère personnel sont cryptées dans la base de données
  • Aucune donnée sensible telle que votre numéro de sécurité sociale ou encore un numéro de carte bancaire n'est enregistrée ou même demandée

3. Utilisation de vos données personnelles

Vos données personnelles ne sont en aucun cas ni transmises ni revendues à des tiers. Elles sont uniquement utilisées dans le cadre de la relation commerciale établie entre vous-même et D n' Dev et pour vous procurer des services.

4. Vos droits en tant qu'utilisateur de ce site

Conformément à la loi du 20 juin 2018 relative à la protection des données personnelles 2018, vous disposez d'un droit d'accès, de portabilité, de rectification, d'opposition, de modification, d'effacement de ces informations.

Si vous souhaitez exercer ce droit, merci d'en formuler explicitement la demande par courriel à l'adresse david@dndev.fr
